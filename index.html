LELLO.JS

What is lello.js?

Lello.js is a set of image processing algorithms implemented in JavaScript & P5. This project is still under construction and more algorithms will be included as soon as possible.

Algorithms

  • Original
  • Addition
  • Substraction
  • Multiplication
  • Threshold
  • Invert
